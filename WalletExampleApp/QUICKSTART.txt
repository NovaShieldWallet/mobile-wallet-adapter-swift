HOW TO RUN THE EXAMPLE APP
===========================

1. Open Package.swift in Xcode:
   cd /Users/ceo/mwa/mobile-wallet-adapter-swift
   open Package.swift

2. Create iOS App Target:
   - File → New → Target
   - iOS → App
   - Name: WalletExampleApp

3. Add Package Dependency:
   - Project → Package Dependencies → +
   - Add Local: ../mobile-wallet-adapter-swift
   - Select: MobileWalletAdapterSwift

4. Copy Source Files:
   - Add all .swift files from:
     WalletExampleApp/Sources/WalletExampleApp/
   - To your new app target

5. Run (⌘R) on iOS Simulator or device

That's it!
